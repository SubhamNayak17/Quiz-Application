# 📝 Interactive JavaScript Quiz App

A dynamic, responsive Quiz Application built using vanilla web technologies. This project tests user knowledge through multiple-choice questions and provides real-time feedback on performance.

---

## 🚀 Features
* **Dynamic Questions:** Questions are rendered dynamically using JavaScript DOM manipulation.
* **Real-time Scoring:** Instant feedback and a final score calculation at the end of the quiz.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using CSS Media Queries.
* **Interactive UI:** Smooth transitions and hover effects for an engaging user experience.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Scripting:** Vanilla JavaScript (ES6+)
